
# NextFolio - 学生数字成长档案

**NextFolio** 是一个专为**中小学生**设计的个人作品集与成长记录工具。我们致力于帮助学生和家长用最简单的方式，建立一个精美的个人主ye，记录学习生活中的每一个闪光时刻。

无论是用于**升学申请**、**社团面试**，还是单纯地**记录成长**，NextFolio 都是你的最佳助手。

![Project Preview](https://picsum.photos/seed/student/800/400) <!-- 建议您后续替换为实际截图 -->

## 🌟 为什么选择 NextFolio？

现在的学生多才多艺，参加比赛、做手工、做公益、学编程......但这些宝贵的经历往往散落在手机相册或纸质证书里，难以整理和展示。

NextFolio 解决了这个痛点：
1.  **零门Tk**：不需要懂任何代码，像填表一样简单。
2.  **集中管理**：把所有的奖状、照片、视频、经历都在一个地方整理好。
3.  **随时展示**：生成一个链接，发给老师、招生官或亲友，让他们在手机或电脑上随时查看。

## ✨ 核心功能

### 📱 漂亮的个人主页 (Front End)
- **专为学生设计**：界面干净、现代，突出个人风采。
- **多主题切换**：支持“现代黑”、“经典白”、“创意波普”等风格，展现个性。
- **多语言支持**：支持中文和英文，方便申请国际学校或对外交流。
- **长图分享模式**：特有的截图模式，一键生成适合发朋友圈或打印的长图简历。
- **全方位记录**：
  - **个人简介**：我是谁，我的爱好是什么。
  - **闪光经历**：在学校担任的职务、参加的活动。
  - **作品展示**：科技小发明、绘画作品、书法展示（支持上传图片和视频链接）。
  - **技能特长**：钢琴十级、编程入门、擅长演讲等。

### ⚙️ 简单的管理后台 (Admin)
- **家长也能轻松上手**：可视化的后台，点点鼠标就能修改内容。
- **AI 帮写助手**：不知道自我介绍怎么写？内置 Google Gemini AI，输入几个关键词，自动帮你润色出彩的文案。
- **即时保存**：
  - **默认模式**：数据保存在你的浏览器里，安全、隐私，无需注册即可体验。
  - **云端模式**：(高级功能) 配置数据库后，数据可云端同步，永不丢失。

## 🚀 快速开始

如果你懂一点技术，可以这样运行本项目：

1. **下载项目**
   ```bash
   git clone https://github.com/your-username/nextfolio.git
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动**
   ```bash
   npm run dev
   ```

4. **开始使用**
   - 访问 `http://localhost:5173/` 查看你的主页。
   - 点击页面底部的 **Admin** 或访问 `http://localhost:5173/#/admin` 进入后台编辑。

## 🛠 技术支持 (致开发者家长)

本项目基于现代 Web 技术构建，如果您是懂技术的家长，可以进行更多定制：
- **React 18** & **TypeScript**: 稳定、快速。
- **Tailwind CSS**: 易于修改样式。
- **Supabase**: 如果需要云端存储，可以直接对接 Supabase 数据库。
